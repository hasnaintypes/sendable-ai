# Welcome to Sendable

Sendable is an AI-powered email outreach platform. Generate personalized cold emails,
automate research, and streamline follow-ups for sales, recruiting, and networking.

## Key Features

- **AI Email Generation** — Craft personalized cold emails with context-aware AI.
- **Secure Authentication** — Email/password, magic links, 2FA, and session management.
- **Dashboard** — Manage campaigns, view analytics, and configure settings.
- **Email Templates** — Responsive templates powered by React Email.

## Quick Start

```bash
git clone https://github.com/your-org/sendable-ai.git
cd sendable-ai
pnpm install
pnpm dev
```

See the [Getting Started](/docs/getting-started) guide for full setup instructions.
